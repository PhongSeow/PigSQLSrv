# PigSQLSrv
#### [English Readme](https://github.com/PhongSeow/PigSQLSrv/blob/main/README.md)
PigSQLSrv 专门用于 SQL Server .net 开发的工具集类库。

## ***目录和文件描述***

### Release

发布执行码目录，如果你不想看到源程序，你可以直接使用这个目录中的文件。

##### Release\DotNet
类库的DLL文件和 NuGet 包。

#### Release\DotNet\PigSQLSrvLib
专用用于 Windows 平台的版本，对 Windows 平台适配性更强。

### Src

源码目录。

#### Src\DotNet\PigSQLSrvLib

类库目录
